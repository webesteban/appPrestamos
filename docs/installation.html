Installation

Prerequisites

To get started with Osen, please follow the steps below to install and set up all the necessary prerequisites:

  • Node.js

    To use build tools and take full advantage of customization options, you will need to download and install Node.js.

  • Ruby 3.2 or newer

    Ruby is required for running various development tools and scripts. Make sure you have latest version of Ruby installed.

  • Rails 8.0.0 or newer

    Rails is also required to run application. Ensure you have Rails installed, along with the required Ruby version.

Installation

You can run this app using any following package manager:

Bun: npm i -g bun or sudo npm i -g bun

Yarn: npm i -g yarn or sudo npm i -g yarn

NPM: Comes pre-installed with Nodejs

Command Description
npm i or yarn or bun i This would install all required dependencies.
bundle install This would install all required Ruby on Rails dependencies.
bin/rails server or rails s Runs the project locally, starts the development server.
